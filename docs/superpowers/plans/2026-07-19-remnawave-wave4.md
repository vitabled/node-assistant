## Закреплённые решения (Alignment)

- **Хостинги** — **новый независимый раздел** (не связан с инфра-биллингом; опц. линк карточки к провайдеру
  инфра-биллинга). Карта — **оффлайн-вектор** (react-simple-maps/d3-geo + топоjson, без внешних тайлов),
  **упор на анимации через `motion`**. Локация карточки = город+страна → координаты из **встроенного оффлайн
  world-cities датасета** (фолбэк — ручной ввод lat/lon). Тоггл видимости континентов + масштаб.
- **Автодетект** — расширить `/api/node/detect`: помимо компонентов читать текущие значения (SSH-порт,
  remnanode_port/xhttp_path/token, открытые порты, домен/серт, WARP/оптимизации) → фронт **предзаполняет форму**
  + помечает совпавшие компоненты «пропустить».
- **E5** — лёгкий скрейп метрик панели (порт 3001, basic-auth), вывод на дэшборде/в статистике; без Grafana-стека.
- **E6 Certwarden** — placement **выбирается оператором**: сервер на боксе панели ИЛИ выделенный; ноды/панель —
  клиенты, авто-рестарт контейнеров при renewal.
- **E7 replace-domain** — визард для **панели и ноды**.
- **E8 Netbird** — **self-hosted control plane** (management/signal/coturn/dashboard), агенты на нодах+панели.
